# 🌦️ Weather Forecast using LSTM

## 📌 Project Overview
This project implements a **Weather Forecasting System** using **Long Short-Term Memory (LSTM)**, a type of Recurrent Neural Network (RNN).  
The model predicts future weather conditions such as temperature, humidity, and pressure based on historical time-series data.

---

## 🚀 Features
- 📊 Time-series forecasting using deep learning  
- 🔁 Sequential data modeling with LSTM  
- 📈 Visualization of actual vs predicted values  
- ⚡ Scalable and customizable model  

---

## 🧠 Technologies Used
- Python  
- TensorFlow / Keras  
- NumPy  
- Pandas

- Scikit-learn  

---

##🔄 Model Workflow

- Data Collection (historical weather data)
- Data Preprocessing (handling missing values, normalization)
- Train-Test Split
- LSTM Model Building
- Model Training
- Prediction
- Visualization of Results

##🧩 LSTM Model Architecture
- Input Layer
- LSTM Layer(s)
- Dropout Layer (optional)
- Dense Layer (Output)


##📊 Results
The model successfully learns patterns from historical data
Provides accurate short-term weather predictions
Output is visualized using graphs comparing actual vs predicted values
 
##✅ Advantages
Handles time-series data efficiently
Captures long-term dependencies
Higher accuracy compared to traditional models

##⚠️ Limitations
Requires large dataset for better performance
Computationally expensive
Sensitive to hyperparameters
